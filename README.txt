This is an application for reserve copying important files and store them in zip archive.

You need to choose files and directories you want to backup and choose the directory where you wish to store your zip-archived data.
In this directory will be created new directory named by "year|month|day" where will be stored archive named by "hour|min|sec"  

Also you can add a comment to the name of the future zip-archive. In this case file name will be "hour|min|sec + comment".
All spaces in the name of file will be replaced by underscores.
